🌐 Liza Rodriguez — Streamlit Portfolio

A modern multipage portfolio web app built with Python and Streamlit.
This project showcases my profile, skills, projects, and contact page with a clean grey theme and optional dark mode.

✨ Features
🌙 Dark Mode toggle on the home page
🎨 Custom CSS styling (grey theme, cards, shadows)
🧭 Multipage navigation (Home, About, Skills, Projects, Contact)
🖼 Profile section with image
📊 Skills with progress bars
📂 Projects displayed using expanders
📞 Contact form UI with social links
🛠 Built With
Python
Streamlit
HTML & CSS (inside Streamlit markdown)
